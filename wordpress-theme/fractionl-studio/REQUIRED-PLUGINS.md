# Required plugins

## Required

- **Advanced Custom Fields Pro** — options page, structured fields, relationships, and the controlled ACF block library.

## Not required

- No page builder
- No form plugin while Tally is retained
- No custom-post-type plugin
- No redirect plugin for the approved URL structure
- No SEO plugin; the theme includes the required metadata, although a supported SEO plugin can take over later

An SMTP plugin is only relevant if native WordPress email functionality is added in the future. The current Tally workflow does not depend on `wp_mail()`.
